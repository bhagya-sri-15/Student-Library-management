# 📚 Student Library Management System

A simple and interactive **Student Library Management System** developed using **HTML, CSS, and JavaScript**. This project helps manage library operations such as adding books, issuing/returning books, and maintaining student records.

---

## 🚀 Features

* 📖 Add and manage books
* 🔍 Search books by title/author
* 🧑‍🎓 Manage student details
* 📤 Issue books to students
* 📥 Return books
* 🗑️ Delete records
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Functionality and logic
